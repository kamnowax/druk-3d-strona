# Strona o druku 3D

Prosta strona internetowa stworzona w HTML, CSS i JavaScript jako projekt zaliczeniowy.

## 📌 Funkcjonalności
- Strona główna (opis technologii druku 3D)
- Galeria drukarek 3D
- Formularz kontaktowy
- Osadzony film (YouTube)
- Responsywny layout (mobile friendly)
- Efekty CSS i proste elementy JavaScript

## 🛠 Technologie
- HTML5
- CSS3 (Flexbox)
- JavaScript

## 📂 Struktura projektu
- index.html – strona główna i prosty film opisujący technologie
- galeria.html – galeria zdjęć 
- kontakt.html – formularz
- style.css – stylowanie
- script.js – JavaScript
- images/ – zdjęcia
